# Лабораторна робота №1. Фронтенд
## Варіант №4
**Тема:** Дошка оголошень групи
**Ключова сутність:** Оголошення
## Функціонал
* Додавання нових оголошень.
* Відображення списку оголошень у таблиці.
* Можливість зміни та видалення оголошень із списку
* Пошук оголошень за заголовком 
* Фільтрація оголошень за категорією
## Запуск
Python:  
python -m http.server 8080

http://localhost:8080/frontend/
